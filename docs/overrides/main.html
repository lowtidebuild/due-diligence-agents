Up and running in minutes

Try it on a
real data room today

No vendor account, no SaaS signup. Install, point it at your data room, run.

# Install
$ pip install dd-agents
 
# Set your API key (Anthropic, Bedrock, or any gateway)
$ export ANTHROPIC_API_KEY="sk-ant-..."
 
# Scaffold a deal config from your data room
$ dd-agents init --data-room ./your_data_room
 
# Run the full 38-step pipeline
$ dd-agents run deal-config.json
